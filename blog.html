QQuestion-3: What is event delegation in the DOM?

Event delegation is a way of handling events more effectively. Without adding event listeners to many child elements, we can add one event listener to parent element. When we click on a child element, the event goes up to the parent. It is more faster process and it works even when new child elements are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process when an event starts from the clicked element and moves up through its parent elements one by one, until it reaches the top
